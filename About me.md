الملخص:
مطور Full Stack محترف مع خبرة شاملة في تطوير تطبيقات الويب وتصميم واجهات المستخدم الرسومية. ملم بتقنيات البرمجة والتصميم الحديثة، وقادر على توفير حلول فعّالة لتحقيق أهداف العمل. يتميز بالقدرة على العمل بفاعلية كاملة في جميع جوانب تطوير البرمجيات.

المهارات:
Full Stack Development:
لغات البرمجة: JavaScript , Python , HTML, CSS
قواعد البيانات: MySQL
Web Development:
تصميم وتطوير مواقع الويب الديناميكية والجذابة.
إنشاء تجارب مستخدم رائعة باستخدام تقنيات تكنولوجيا الويب الحديثة.
GUI Development:
تصميم وتنفيذ واجهات مستخدم جذابة باستخدام مكتبات وأدوات مثل tkinter.
تحقيق تكامل فعّال بين الواجهة والتطبيقات الخلفية.
التعليم:
أدرس في مدرسة أحمد ضيف الله الدولية للتكنولوجيا التطبيقية والذكاء الاصطناعي

تصميم وتنفيذ قواعد البيانات باستخدام MySQL.
تحسين أداء التطبيقات وتحسين تجربة المستخدم
تصميم وتنفيذ واجهات مستخدم باستخدام tkinter.
تحسين تجربة المستخدم وتكامل الواجهة مع البرمجيات الأخرى.
